# 부분집합(비트연산자 이용)

```python
T = int(input())
A = [i for i in range(1, 13)]

for tc in range(1, T+1):
    N, K = map(int, input().split())

    B = len(A)
    cnt = 0

    for i in range(1<<B):
        A_list = []
        sum_A = 0
        for j in range(1+B):
            if i & (1<<j):
                A_list.append(A[j])
        if len(A_list) == N:
            for j in range(len(A_list)):
                sum_A += A_list[j]
            if sum_A == K:
                cnt += 1


    print(cnt)
```
