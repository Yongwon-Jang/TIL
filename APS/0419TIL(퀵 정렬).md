# QuickSort

- 분할정복 기법과 재귀 알고리즘을 이용한 정렬 알고리즘
- 평균 시간 복잡도는  `O(nlogn)`, 최악의 경우`O(N^2)`
- 분할 방법에 따라 Hoare, Lomuto 방법이 존재한다. 더 나은 것을 선택해서 사용하면 됨
- 코드

```python
def quicksort(A, l, r):
    if l < r:
        s = partition(A, l, r)
        quicksort(A, l, s-1)
        quicksort(A, s+1, r)

# Hoare 파티션
def partition(A, l, r):
    p = A[l]
    i = l + 1
    j = r
    while i <= j:
        while i <= j and A[i] <= p:
            i += 1
        while i <= j and A[i] >= p:
            j -= 1
        if i <= j:
            A[i], A[j] = A[j], A[i]
    A[l], A[j] = A[j], A[l]
    return j


# Lomuto 파티션
def partition(A, l, r):
    x = A[r]
    i = l - 1
    for j in range(l,r):
        if A[j] <= x:
            i += 1
            A[i], A[j] = A[j], A[i]
    A[i+1], A[j] = A[j], A[i+1]
    return i + 1


arr = [1, 45, 22, 81, 23, 34, 99, 22, 17, 8]
print(arr)
quicksort(arr, 0, len(arr)-1)
print(arr)
```
