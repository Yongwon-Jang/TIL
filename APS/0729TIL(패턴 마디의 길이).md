# SWEA 2007. 패턴마디의 길이

패턴이 반복되는 글자의 길이를 구하는 문제

- 내가 푼 코드

```
T = int(input())
for tc in range(1, T+1):
    string = input()

    idx = 0
    for i in range(1, len(string)):
        if string[i] == string[0] and string[i-1] == string[-1]:
           idx = i
           break

    print('#{} {}'.format(tc, idx))
```
