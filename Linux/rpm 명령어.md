## rpm 명령어

> rpm은 보통 yum으로 설치하기 어려운 상황이나 yum repository에 등록되어 있지 않은 프로그램들을 다운받는 상황에 사용합니다.

#### RPM 패키지 설치 명령어

- 패키지 설치

```
[root]# rpm -ivh [패키지명]
```

- 패키지 업데이트

```
[root]# rpm -Uvh [패키지명]
```

- 패키지 설치 확인

```
[root]# rpm -qa [패키지명]
```



#### RPM 패키지 삭제

- `rpm -qa [패키지명]` 으로 설치된 패키지를 확인하고 `-e` 옵션을 이용하여 삭제 할 수 있다.

```
[root]# rpm -e [삭제할 패키지명]
```

- 삭제가 정상적으로 안되고 `dependency` 관련 오류가 날때

```
[root]# rpm -e [삭제할 package명] --nodeps
```
