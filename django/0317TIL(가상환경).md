# django 가상환경

```bash
$ python -m venv venv

$ pip list

$ source venv/Scripts/activate

$ pip install django


.gitignore 생성
README.md 생성
$ django-admin startproject crud .

$ pip freeze

$ pip freeze > requirements.txt

# rqquirements에 있는거 다 설치 해줘
$ pip install -r requirements.txt


# 활성화 끄는법
$ deactivate
```
